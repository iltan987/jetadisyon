# JetAdisyon

JetAdisyon is a Windows desktop application for Turkish restaurant owners who manage orders across multiple food delivery platforms — Yemeksepeti, Trendyol Go, Getir, and Migros Yemek — simultaneously.

Instead of watching four separate tablets and manually accepting each order, restaurant owners run JetAdisyon on a single PC. The app connects to all platforms via their official Partner APIs, auto-accepts incoming orders within seconds, plays a sound alert, and prints a ticket to the kitchen printer — automatically, without any manual interaction.

> 🚧 This project is currently in early development.

## Tech Stack

- **Desktop:** Electron
- **Backend:** NestJS (local server)
- **Frontend:** React + Vite
- **Database:** SQLite (local) + Supabase (license & auth)
- **Language:** TypeScript
